#1
  CREATE DATABASE
 Run database_setup.sql in MySQL
 Creates 'student_db' database and 'students' table

#2
  ADD JDBC DRIVER
 Download mysql-connector-java-x.x.x.jar
 Copy to WebContent/WEB-INF/lib/

#3
  UPDATE CREDENTIALS
 Edit src/com/student/servlet/StudentDAO.java
 Set: jdbcUsername, jdbcPassword, jdbcURL

#4
  BUILD PROJECT
 Mac/Linux: ./build.sh
 Windows: build.bat
 Or manually: javac -d build/classes src/com/student/servlet/*.java

#5
  DEPLOY & RUN
 Copy WebContent to: $TOMCAT_HOME/webapps/StudentManagementSystem/
 Start Tomcat: ./startup.sh (or startup.bat)
 Open: http://localhost:8080/StudentManagementSystem/

If you encounter issues:

1. Check Tomcat logs:
   Mac/Linux: tail -f $TOMCAT_HOME/logs/catalina.out
   Windows: View catalina.out in logs folder

2. Check MySQL connection:
   mysql -u root -p -e "SHOW DATABASES; USE student_db; SHOW TABLES;"

3. Verify file locations:
   Check all files are copied to correct Tomcat directories

4. Clear browser cache:
   Ctrl+Shift+Delete (or Cmd+Shift+Delete on Mac)

5. Review documentation:
   Check README.md, SETUP_GUIDE.md, DEPLOYMENT_GUIDE.md
